# PixelPop Market

## Demo: 
https://pixelpop-market.netlify.app/

PixelPop Market is a modern e-commerce-style web application featuring a vibrant, pixel-inspired design. It allows users to browse products, view details, and interact with a clean, responsive interface built for a smooth shopping experience.

---

## Features

- Browse a collection of products
- View detailed product information
- Add items to cart
- Dynamic cart updates
- Pixel-themed UI with modern styling
- Fully responsive design

---

## Built With

- HTML5
- CSS3
- JavaScript (ES6+)
- Modular component structure

---

## Overview

PixelPop Market focuses on combining a playful pixel aesthetic with real-world e-commerce functionality. The app demonstrates:

- Clean layout and product grid design  
- Interactive cart system  
- Smooth UI transitions and responsiveness
